## [v1.0.0] - 2026-05-22

### Added

- Initial Release: Core functionality to generate ASCII tree diagrams and structures from text/Markdown.
- Export feature allowing users to download structured `.zip` archives containing generated `.txt`, `.html`, or `.md` files.
- Dynamic user interface with Theme, Display, and Format settings saved via Local Storage.
- Find & Replace text manipulation system with customizable rules and quick presets.
- Advanced Markdown parsing (supports headers and numbered/bulleted lists).
- Comment preservation system that exports aligned metadata into standalone `_comments.txt` files.
